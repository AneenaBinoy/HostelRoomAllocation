 Problem Statement

   Managing hostel room allocation manually is error-prone. This project automates room allocation, viewing, updating, and vacating.


 Objective

    Maintain student records and room allocation

    Automate CRUD operations with GUI
  
    Connect to MySQL database



 Features

    Add, View, Update, Vacate student rooms

    GUI using Java Swing

    Database connection via JDBC



Technologies Used

   Java 8+

   Java Swing

   MySQL / XAMPP

   JDBC

   IntelliJ IDEA


   How to Run

    Clone repo: `git clone https://github.com/Aneena Binoy/HostelRoomManagement.git`

    Start XAMPP (Apache + MySQL)

    Run `hostel_db.sql` in phpMyAdmin to create DB \& table

    Open project in IntelliJ

    Add MySQL JDBC connector in `lib/` folder

    Run `MainMenu.java` to start GUI
